## Чек-лист Рекомендации Стратегия
>[!IMPORTANT]
> build: 0.1.2.3\
> Дата проверки: 21.05.2026\
> Окружение: Windows 11, Chrome\
> Проверяющий: Намазов Максим
- Все поля сгенерированы в соответствии с заданной тематикой при Опросе $${\color{green}+}$$

- Диалог выбора соц.сетей открывается и выбор соц. сетей сохраняется $${\color{red}-}$$
	- Для того, чтобы закрыть окно выбора соц. сетей, требуется его полностью прокручивать вниз, что не удобно. Возможно стоит зафиксировать кнопки "Отмена" и "Сохранить" на границе окна. $${\color{purple}remark}$$
	- Выбор соц. сетей работает корректно, все проставленные галочки визуально сохраняются, однако json strategy остаётся незменным - там по ключу social всегда отображаются только 4 соц. сети: VK, TG, LI(LinkedIn), IG(Instagram). $${\color{red}bug}$$ (cм. bug-report: Ответы во вкладке Мой бренд -> Форматы соц. сети ответы сохраняются только визуально, но json strategy не перегенерируется)

- Диалог выбора роли открывается $${\color{green}+}$$
	- Смена роли на "Новичок" сопровождается статус кодом ответа 422 Unprocessable Entity на PATCH запрос. Справа снизу высвечивается попап "Не удалось сохранить изменения" Ошибочный ответ не сохраняется. $${\color{red}bug}$$ (*см. bug-report:* **Невозможно сменить роль на Новичок**)

- Редактируемые поля возможно отредактировать и их значения обновляются $${\color{green}+}$$
	- Окно Редактирования, расширяясь, увеличивает размер страницы, что не удобно, если пользователь захочет ввести большой текст. Возможно стоит сделать скроллер в рамках фиксированного окна редактирования $${\color{purple}remark}$$

- Поля для которых есть функция повторной генерации, возможно перегенрировать $${\color{red}-}$$
	- При попытке перегенерировать содержимое любых полей во вкладке Рекомендации\Стратегии(Мой бренд) возникает ошибка. Всплывает попап. $${\color{red}bug}$$  (*см. bug-report:* **Невозможно перегенерировать поля с функцией повтороной генерации в Рекомендации_Статегии**)

- Сегменты аудитории возможно выбрать $${\color{green}+}$$

- Возможна повторная генерация нескольких полей одновременно $${\color{red}-}$$
