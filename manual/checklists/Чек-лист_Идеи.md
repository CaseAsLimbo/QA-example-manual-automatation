## Чек-лист Идеи
- Имеющиеся идеи показываются списком $${\color{green}+}$$

- Создать и сохранить идею собственноручно возможно $${\color{green}+}$$
	- При быстром нажатии на кнопку "Сохранить" для создания идеи, в список добавится несколько одинаковых идей. $${\color{red}bug}$$ (*см. баг-репорт:* **Быстрое нажатие на кнопку Сохранить при создании идеи создает несколько одинаковых идей** )
	- Можно сохранить пустую идею(без заголовка и описания) $${\color{purple}remark}$$

- Удалить идею из списка имеющихся возможно $${\color{green}+}$$

- Идея при использовании отображается в черновике статьи $${\color{green}+}$$
	- Нет ограничения на поля описания и заголовка идеи в черновике статьи $${\color{purple}remark}$$
	- Кнопка "Отменить" при редактуре статьи не отменяет сделанные действия, и всегда перемещает во вкладку Редактор(общее меню блога), даже если в вкладку редактуры конкретной статьи мы попали из вкладки Идеи  $${\color{purple}remark}$$
	- В редактуре статьи можно сохранить, а затем опубликовать или добавить в архив пустую статью $${\color{purple}remark}$$
	- Во вкладке Редактор переключение между статусами статей не отображается визуально - всегда подсвечен только заголовок "Все статьи" $${\color{purple}remark}$$

- После сохранения черновика статьи идея отображается как использованная  $${\color{red}-}$$
	- Использованная идея в том же виде отображается в списке идей, нет никакого визуального маркера о том, что она использована. При нажатии на кнопку "В черновики" для использованной идеи возвращается код ошибки 500, в левом нижнем углу отображается попап "Ошибка при изменении статьи". $${\color{purple}remark}$$

- Меню создания идеи мануально работают корректно  $${\color{red}-}$$
	- Нет ограничения на длину заданного заголовка идеи $${\color{purple}remark}$$
	- Перенос слов заголовка на следующую строку происходит только при наличии пробельных символов между словами. При написании строки без пробельных символов длиной большей, чем текстовая рамка, символы в этой строке выходят за пределы текстового блока(более 28 символов.) $${\color{red}bug}$$ (*cм. видео:* **Перенос слов заголовка на следующую строку происходит только при наличии пробельных символов между словами.**)
	- Боковое меню редактирования идей частично перекрывает созданные идеи. $${\color{purple}remark}$$

## Дополнительно
- Лимит в 50 идей прописанный в json `ideas?status=all&offset=0&limit=50` можно превысить $${\color{red}bug}$$ (*см. видео:* **Лимит в 50 идей можно превысить**)
